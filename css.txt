	11. display | visibility
		
		display : none;  invisible && 자리차지 x
		visibility : hidden; invisible && 자리는차지함.

	12. position
		
		1) static(default)

		2) relative : 관계형이므로, 다른 엘리먼트를 기준으로 위치이동.

		3) fixed : 현재 화면 내에서 절대위치

		4) absolute : 전체내(상위 태그를 가리킴)에서의 절대위치
	
		** 부모가 relative면 자식은 absolute로 하는 게 정석.

	13. z-index
		
		좌표상 z축 의미, 높을수록 화면 앞으로, 낮을수록 화면 안으로

		0.0~1.0 사이의 값, 투명도
