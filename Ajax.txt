동기 방식

	client와 server가 동기화되어 있어서 client의 요청에 대해 server가 처리하는동안

	client는 다른 작업을 수행할 수 없다.


비동기 방식

	client와 server가 독립적으로 작업을 수행할 수 있다.

	한 페이지 내에 특정 영역만을 갱신하기 위해서 사용되는 경우도 있음. 
	
	(서버가 응답을 보내면 페이지의 모든 영역이 갱신되지만, 특정 경우엔 일부 영역만 갱신되기만 하면 되는 경우도 있음)

	eg. naver의 실시간 검색 탭 (페이지의 나머지 영역은 변화하지 않지만, 실시간검색 탭만 변화함. 이런 경우, 굳이 페이지 전체를 리로딩시킬 필요가 없다는 것) 
	


이러한 비동기 방식의 페이지처리를 위해 활용하는 기술?이 Ajax이다.

엄밀히 말해서, java script를 통해서 서버에 데이터를 요청하는 것이다. 중요한 것은 동기방식이 아닌, 비동기방식이라는 것.
