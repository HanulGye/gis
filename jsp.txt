=================================== JSP ===================================

Java Server Page
- HTML내에 자바코드 쓸 수 있게 만든 언어

1. 스크립트 요소
	1) 스크립트리			:<% 자바코드 %>
	2) 표현식(일종의 화면 출력)	:<%= 데이터 %>

2. 내장 객체

	: Container(TomCat)에서 생성되는 객체

	1) Request, Response
		a) Servlet : 매개변수로 받아서 사용
		b) jsp     : 참조변수 사용(request, response)
	


** jsp에는 HTML코드, java코드가 존재하는데, 읽어내는 순서는 java가 우선이다. 왜냐면 back-end에 해당하니까.
